# 🛠️ Day 4: Advanced HTML Elements

## 📋 Lists

### 🔢 Ordered Lists
```html
<ol>
  <li>First 🥇</li>
  <li>Second 🥈</li>
</ol>
```

### ✔️ Unordered Lists
```html
<ul>
  <li>Item A ✅</li>
  <li>Item B ✅</li>
</ul>
```

### 📑 Description Lists
```html
<dl>
  <dt>Term 📚</dt>
  <dd>Description 🗒️</dd>
</dl>
```

## 📊 Tables

- Structured data display.

### Example
```html
<table>
  <thead>
    <tr><th>Header 1 📌</th><th>Header 2 📌</th></tr>
  </thead>
  <tbody>
    <tr><td>Data 1 📎</td><td>Data 2 📎</td></tr>
  </tbody>
</table>
```
